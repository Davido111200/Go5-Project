# Dev guide

1. cd into your dir
```bash
# for example
cd text_detection
```

2. create venv
```bash 
python3.9 -m venv env
. env/bin/activate
```

3. install requirements
```bash 
pip install -r requirements.txt
```

4. dev

on your Python file.
