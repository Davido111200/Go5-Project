# how to start

1. clone 
```bash
git@github.com:phamquiluan/Go5-Project.git
```
2. pin up services
```bash
cd Go5-Project

export DOCKER_BUILDKIT=1
docker compose up
```

3. open browser with below url

```bash
localhost:8501  # you can update the port the .env file
```
